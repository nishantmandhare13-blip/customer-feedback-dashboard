# 📊 Customer Feedback Analysis Dashboard

A modern, interactive Streamlit dashboard for analyzing customer feedback using data visualization, sentiment analysis, and business insights.

**Streamlit App:**

## 📌 Features

* 📂 Upload customer feedback CSV files
* 📊 Interactive Plotly visualizations
* ⭐ Rating distribution analysis
* 😊 Sentiment analysis dashboard
* 📈 Monthly feedback trends
* 🌍 Region-wise filtering
* 📋 Data preview and summary statistics

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/customer-feedback-dashboard.git
```

Navigate to the project:

```bash
cd customer-feedback-dashboard
```

Create a virtual environment:

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
